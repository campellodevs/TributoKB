# 🏀 Tributo a Kobe Bryant

Projeto desenvolvido como uma página de tributo ao jogador de basquete **Kobe Bryant**, com foco em semântica HTML, estilização moderna em CSS e interações simples em JavaScript.

O objetivo deste projeto é demonstrar habilidades em **estruturação de páginas web**, **design visual**, **responsividade** e **experiência do usuário**, servindo como peça de portfólio.

🔗 **Veja o projeto em produção:**  
https://campellodevs.github.io/TributoKB/

---

## ✨ Funcionalidades

- Layout semântico utilizando HTML5 (`header`, `main`, `section`, `footer`)
- Estilo visual inspirado em **polaroid**, com foco estético e memorial
- Fundo cinza granulado para dar identidade visual
- Linha do tempo interativa:
  - Ao passar o mouse sobre os anos, informações históricas são exibidas dinamicamente
- Design responsivo (mobile-first)
- Código organizado e de fácil leitura

---

## 🛠️ Tecnologias Utilizadas

- **HTML5**
- **CSS3**
  - Flexbox
  - Animações e transições
  - Media queries
- **JavaScript (Vanilla)**
  - Manipulação do DOM
  - Eventos de interação (`hover`)
